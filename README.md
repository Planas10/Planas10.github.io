## ¿Que es HTML?¿CSS?¿Javascript?

### HTML:
HTML (Hypertext Markup Language) es un lenguaje de marcado utilizado para crear y diseñar páginas web. Permite la estructuración y organización del contenido mediante etiquetas y atributos. HTML se utiliza en conjunto con CSS y JavaScript para crear páginas web interactivas y dinámicas. Los navegadores web interpretan el código HTML y lo muestran como páginas web para el usuario final.

### CSS:
CSS (Cascading Style Sheets) es un lenguaje de diseño utilizado para dar estilo y presentación visual a las páginas web. Permite modificar el aspecto visual de elementos HTML, como colores, fuentes, tamaños, márgenes, entre otros.
CSS se utiliza en conjunto con HTML y JavaScript para crear páginas web interactivas y dinámicas. Con CSS se pueden crear diseños adaptativos a diferentes dispositivos y tamaños de pantalla. Los navegadores web interpretan el código CSS y lo aplican al contenido HTML para mostrar páginas web estilizadas y atractivas.

### Javascript:
JavaScript es un lenguaje de programación que se utiliza para crear páginas web interactivas y dinámicas. Se ejecuta en el lado del cliente, es decir, en el navegador web del usuario final, y permite crear efectos visuales, animaciones, validar formularios y realizar operaciones complejas en el cliente.
También se puede utilizar en el lado del servidor para crear aplicaciones web completas. JavaScript se integra con HTML y CSS para crear aplicaciones web modernas y potentes. Es un lenguaje versátil y ampliamente utilizado en el desarrollo web y de aplicaciones.

## ¿Que es el formato MarkDown?
Markdown es un lenguaje de marcado ligero utilizado para dar formato a textos en la web de manera sencilla y rápida. Permite crear títulos, párrafos, listas, tablas, enlaces, imágenes, entre otros elementos. El formato se realiza utilizando símbolos y caracteres especiales en el texto plano.

## ¿Que es el formato JSON?
JSON (JavaScript Object Notation) es un formato de intercambio de datos utilizado en aplicaciones web y servicios web. Es un formato de texto ligero que utiliza una sintaxis sencilla para representar objetos y datos estructurados. Los datos se almacenan en pares de clave-valor y se pueden anidar para crear estructuras más complejas.
JSON es fácil de leer y escribir para humanos y es compatible con una amplia variedad de lenguajes de programación. Es ampliamente utilizado en la comunicación de datos entre aplicaciones cliente-servidor y en la integración de sistemas.

## ¿Como se convierte un objeto a JSON?
Para convertir un objeto a JSON, se utiliza la función `JSON.stringify()` en JavaScript. Esta función toma el objeto como parámetro y lo convierte en una cadena JSON. La cadena JSON resultante se puede enviar a un servidor, almacenar en una base de datos o utilizar en otras aplicaciones.
En la conversión, los valores del objeto se convierten en pares clave-valor en formato JSON y se escapan los caracteres especiales.

## ¿Y JOSN a un objeto?
Para convertir una cadena JSON en un objeto en JavaScript, se utiliza la función `JSON.parse()`. Esta función toma la cadena JSON como parámetro y la convierte en un objeto JavaScript. En la conversión, los pares clave-valor en formato JSON se convierten en propiedades y valores del objeto JavaScript.
Es importante asegurarse de que la cadena JSON sea válida antes de intentar analizarla, ya que si contiene errores o está mal formada, la función `JSON.parse()` generará una excepción.
